# Car Service APP

Full-stack web application for managing and browsing car service information. The project includes a Vue.js frontend, Express.js backend, and a database connected through a REST API. Users can view available services, check vehicle service history, and interact with car maintenance data through a modern web interface. The backend handles API requests, business logic, and database operations, while the frontend provides a responsive user interface.

Main features include:

REST API built with Express.js
Frontend built with Vue.js
Database integration for storing vehicles, service records, and maintenance data
CRUD operations for managing vehicles, services, and repair records
Separation of frontend and backend architecture

The project demonstrates full-stack development concepts such as API design, database integration, and client-server communication in the context of a car service management system.
